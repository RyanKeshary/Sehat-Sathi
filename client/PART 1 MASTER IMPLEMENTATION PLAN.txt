1.4 вЂ” Critical Architecture Notes for AI Builder
text

OFFLINE-FIRST RULE: Every data write must go to IndexedDB first,
then sync to server. Never block UI on network requests.

SECURITY RULE: All PHI (Protected Health Information) in IndexedDB
must be encrypted with AES-256-GCM before storage. Use WebCrypto API.

WEBRTC RULE: Always prefer VP9 codec. Start with 480p resolution.
Implement adaptive bitrate. Include TURN server fallback.

LANGUAGE RULE: Every user-facing string must pass through i18n.
Support RTL layout for Urdu. Use Noto Sans Devanagari for Hindi.

ANIMATION RULE: Use will-change: transform sparingly. All scroll
animations via GSAP ScrollTrigger. Page transitions via Framer Motion.

ACCESSIBILITY: WCAG 2.1 AA minimum. All icons have aria-labels.
High contrast ratios (4.5:1 minimum). Voice navigation supported.

TRIAGE COLORS: Green = self-care, Yellow = schedule visit, 
Red = emergency. Never use red for non-emergency UI elements.
